The steps below describe a set of tools you can use to download the UX Aspects package; these are development dependencies only, your application will not require these in production. Install with Bower

  1. Install Node.js, which provides the npm package manager.
  2. Install Bower: npm install -g bower
  3. Install the UX Aspects bower package: bower install ux-aspects#{{ version }}

Building from Source

  1. Clone the GitHub repository.
  2. Install the dependencies: npm install
  3. Build the project: grunt build. The resources (scripts, CSS, fonts, and images) can be found in the dist folder.
