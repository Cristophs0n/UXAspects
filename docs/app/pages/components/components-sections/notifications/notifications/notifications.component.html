Notifications can be used to inform the user of occurring events or to alert the user of something that may require their attention. Notifications will appear in the top right corner of the browser window and can be dismissed by the user by clicking the close button.

Notifications are controlled by the notificationService which can be injected into any controller in the application.

The content of the notification should be specified inside an ng-template element.

The notificationService has several functions available to control notifications.

This function should be called to show a notification. It should be given an ng-template element containing the elements to be displayed and an object containing properties to configure the notification. It will return a NotificationRef object which will contain the information passed to the showNotification function along with a date property which will contain the time at which the notification was shown. This function will return an array of NotificationRef objects. This function can be called to dismiss a notification. It should be passed the NotificationRef object to dismiss. This function will dismiss any currently visible notifications.

The following options can be used to configure the location and behavior of the list of notifications:

Sets the position of the list of notifications within the browser window. The possible options are topRight and topLeft. Sets the order in which notifications are displayed. The possible options are append and prepend. When set to append newer notifications will appear below older ones. When set to prepend newer notifications will appear above older ones
